# Modal de Confirmação

## Objetivo
Evitar ações irreversíveis sem confirmação explícita.

## Usado para
- Aprovar operação
- Rejeitar operação
- Cancelar
- Marcar como financiada

## Componentes
- Título
- Texto explicativo
- Botão: Confirmar
- Botão: Cancelar

## Comportamento Visual
- Apenas fecha ou confirma visualmente
