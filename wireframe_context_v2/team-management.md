# Gestão de Equipe

## Objetivo da Tela
Permitir que o Sacado cadastre e visualize os usuários internos
e defina responsabilidades (rules) de forma clara.

## Usuário
- Sacado

## Perfis de Acesso (apenas visual)
- Admin Financeiro
- Aprovador
- Operador
- Leitor

## Componentes
### Tabela de Usuários
Colunas:
- Nome
- E-mail
- Perfil (badge)
- Status (Ativo / Inativo)

Ações:
- Editar
- Desativar

### Botão Principal
- “Adicionar membro”

### Modal: Adicionar / Editar
- Input: Nome
- Input: E-mail
- Select: Perfil
- Select: Status
- Botões: Salvar / Cancelar

## Comportamento Visual
- Alterações refletem apenas na UI
- Salvar exibe toast estático
