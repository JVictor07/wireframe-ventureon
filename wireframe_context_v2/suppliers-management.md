# Gestão de Fornecedores

## Objetivo da Tela
Permitir que o Sacado gerencie os fornecedores que participam do programa.

## Usuário
- Sacado

## Componentes
### Tabela de Fornecedores
Colunas:
- Nome
- CNPJ

### Botão Principal
- “Adicionar fornecedor”

### Modal: Adicionar / Editar
- Input: Nome
- Input: CNPJ
- Botões: Salvar / Cancelar

## Comportamento Visual
- Sem exclusão no MVP
- Salvar é apenas visual
