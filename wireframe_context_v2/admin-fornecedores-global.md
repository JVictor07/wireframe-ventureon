# Gestão Global de Fornecedores - Admin

## Objetivo da Tela
Dar à equipe da Ventureon (admin) uma visão consolidada de TODOS os fornecedores da plataforma, independente do sacado, para monitoramento e suporte.

## Usuário
- Admin (Ventureon)

## Componentes

### Cards de Métricas
- **Total de Fornecedores**: Quantidade total cadastrada na plataforma
- **Com Acesso Habilitado**: Fornecedores que podem acessar a plataforma
- **Fornecedores Ativos**: Com operações recentes
- **Taxa de Adoção**: Percentual de fornecedores com acesso habilitado

### Tabela Consolidada
Colunas:
- Fornecedor (nome e CNPJ)
- Sacados associados (lista de empresas compradoras)
- Status de acesso (Habilitado/Desabilitado)
- Status de atividade (Ativo/Inativo)
- Última operação (data)
- Total de operações
- Valor total de operações
- Ações (Impersonar, Ver Operações)

### Filtros Avançados
- Busca por nome ou CNPJ
- Filtro por sacado
- Filtro por status de acesso
- Filtro por atividade

## Ações de Suporte
- **Impersonar**: Navega para o dashboard do fornecedor (apenas se acesso habilitado)
- **Ver Operações**: Visualiza histórico de operações do fornecedor
- **Desabilitar Acesso**: Em caso de problemas ou suporte

## Comportamento Visual
- Badges coloridos para status de acesso e atividade
- Múltiplos sacados listados por fornecedor
- Métricas de engajamento e uso da plataforma
- Visão panorâmica para identificar problemas e oportunidades

## Diferencial
Esta tela permite que o admin tenha visão completa de todos os fornecedores, cruzando dados de múltiplos sacados, facilitando suporte e monitoramento da adoção da plataforma.
